# SkillHub API

**SkillHub** — это комплексная бэкенд-платформа для управления образовательным процессом (LMS). Система создана для автоматизации взаимодействия между преподавателями и студентами, управления учебными материалами и контроля успеваемости.

## 🎯 О проекте
SkillHub предоставляет мощное REST API, которое позволяет управлять полным циклом обучения: от записи на курсы до выставления оценок и формирования аналитических отчетов.

## 🛠 Ключевые возможности
* **Управление пользователями:** Безопасная аутентификация и разделение доступа (студенты/преподаватели).
* **Система курсов:** Гибкое создание структуры учебных программ.
* **Учет успеваемости:** Трекинг посещаемости занятий в реальном времени.
* **Процесс зачисления:** Автоматизированная система записи студентов на курсы.
* **Система обратной связи:** Встроенные механизмы для отзывов и подачи жалоб.
* **Управление ресурсами:** Централизованное хранение и доступ к учебным материалам.

## ⚙️ Технический стек
Проект разработан с использованием современных стандартов:
* **Язык:** C#
* **Framework:** ASP.NET Core 8.0
* **База данных:** SQLite / Entity Framework Core
* **Архитектура:** Чистая RESTful архитектура с использованием DTO-моделей.
